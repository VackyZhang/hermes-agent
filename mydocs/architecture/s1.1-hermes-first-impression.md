## 1. 环境准备

- [ ] 克隆仓库：`git clone git@github.com:VackyZhang/hermes-agent.git`
- [ ] 创建虚拟环境：`python3 -m venv .venv`
- [ ] 安装依赖：`pip install -e ".[dev]"`
- [ ] 配置 API Key：`~/.hermes/.env`
- [ ] 验证安装：`hermes --version`
