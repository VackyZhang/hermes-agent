1. **使用搜索功能**：
   - VS Code：`Cmd+Shift+F`（全局搜索）
   - 搜索关键词：`tool`、`loop`、`agent`、`register`

2. **查看目录结构**：
   - 重点关注 `packages/`、`src/`、`tools/`、`agents/` 等目录

3. **阅读 README**：
   - Claude Code 的 README 可能包含架构说明

### A2：如何判断代码的重要性？

1. **看文件名**：
   - `loop.ts`、`agent.ts`、`tool.ts` 等文件名直接指示功能

2. **看函数名**：
   - `registerTool()`、`executeTool()`、`runLoop()` 等函数名直接指示功能

3. **看注释**：
   - 良好的代码会有注释说明设计思路
