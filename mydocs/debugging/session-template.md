# 调试记录模板

复制此模板创建新的调试记录：

```markdown
# 调试记录 — YYYY-MM-DD 标题

## 环境

- 分支：`vacky/dev`
- Commit：`git rev-parse --short HEAD`
- 配置：`~/.hermes/config.yaml` 的关键修改
- 启动命令：`VACKY_DEBUG=1 hermes`

## 目标

本次调试想验证/观察什么？

## 过程

### 步骤 1：...

- 做了什么
- 观察到了什么
- 输出日志/截图

### 步骤 2：...

## 结果

- 成功 / 失败 / 待确认
- 关键发现

## 问题与待办

- [ ] 问题 1
- [ ] 问题 2

## 相关笔记

- `../architecture/xxx.md`
```
